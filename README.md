## RSSフィード

qnypのユーザー毎のRSSフィードは以下のURLで取得できます。

```
https://qnyp.com/ユーザー名/feed
```

フィードには、直近20件の視聴ログについての、

- 対象のエピソード名
- 作品の画像
- 感想本文のHTML

が含まれます。
