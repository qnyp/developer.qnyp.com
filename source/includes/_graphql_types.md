# Object types (オブジェクト型)

## Title (タイトル)

Title型はqnypに登録されている1件のアニメ作品を表現します。

### フィールド

名前 | 型 | 説明
---- | ---- | ----
`id` | [ID](#id) | タイトルを識別するための値
`name` | [String](#string) | 作品の名前
